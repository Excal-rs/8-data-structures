Dictionary

  • Set Value: Associates a value with a specific key in the dictionary.
  • Get Value: Retrieves the value associated with a specific key in the dictionary.
  • Delete Key-Value Pair: Removes a key-value pair from the dictionary based on the specified key.
  • Keys: Returns a list of all the keys in the dictionary.
Unlike sequences (like lists or arrays), which are indexed by a range of numbers, dictionaries are indexed by keys. This means that instead of using numerical indices to access elements, you use keys.
