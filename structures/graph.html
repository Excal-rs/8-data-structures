Graphs

  • Add Node: Adds a node to the graph.
  • Add Edge: Adds an edge between two nodes in the graph.
  • Depth-First Search (DFS): Explores as far as possible along each branch before backtracking in a graph.
  • Breadth-First Search (BFS): Explores all the vertices at the current depth prior to moving on to vertices at the next depth in a graph.
Graphs have a different Time complexity for each different function the graph can do. In order to store the graph, there is an efficiency of O(V+E) - linear. In order to add a vertex the graph it is O(1) at best and O(V^2) at worst and to remove a vertex it is O(E). To add an edge it is O(1) and to remove an edge it is O(V)

The Space complexity is O(n) if the graph isnt used as an array but it is O(1) if it is used as an array
