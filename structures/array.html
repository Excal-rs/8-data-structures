Array

  • Access Element: Retrieves the value of an element at a specific index in the array.
  • Modify Element: Updates the value of an element at a specific index in the array.
  • Array Length: Returns the number of elements in the array.
  • Search Element: Finds the index of a specified element in the array.
Arrays have a time complexity of O(1) at best case when accessing an element by its index, as this operation can be performed instantly. However, the average and worst-case time complexity for operations like searching and removing an element in an array is O(n), as it requires traversing the array linearly. When arrays are multidimensional or nested within other arrays, the time complexity can worsen to O(n^2) at the worst case.

Arrays exhibit a space complexity of O(n) since they store elements contiguously in memory. Arrays are considered a static data structure, meaning they allocate a fixed amount of memory. This can result in a constant space complexity of O(1) when the array size remains constant throughout its lifetime.
