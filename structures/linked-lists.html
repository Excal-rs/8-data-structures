Linked List

  • addItem() - Adds a item at a given location
  • getLen() - Returns length of linked list`
  • search() - Searches for a existing item
  • delete() - Deletes a existing item
  • and more...
Linked lists have a Time complexity of O1 at the best case because the first node could be the node that needs tampering with. However at an average or even at worst, the Time complexity of a linked list is O(n) as the list goes through each node one by one to get to the one that needs changing.

A Linked lists Space complexity as best is O(n) because it is a dynamic data structure, and can only uses memory when needed and the size of the data can grow and shrink where possible. However a linked list can also be used as an array (a static data structure) which means that its Space complexity is O(1) and so will use a constant amount of memory no matter what.
