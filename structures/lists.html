Lists

  • Append: Adds an element to the end of the list.
  • Remove: Removes the first occurrence of a specified element from the list.
  • Sort: Arranges the elements of the list in ascending or descending order.
  • Slice: Extracts a portion of the list, creating a new list.
Lists have a Time complexity of O1 at best case since adding a value to a list can be done instantly, but removing or finding an item in the list has a Time complexity of O(n) since you have to go through the list in a linear fashion. If lists are within other lists, the time complexity goes to O(n^2) at the worst case

Lists have a space Complexity of O(n) as they are stored non-contiguously, and lists are a dynamic data structure, so only take up as much memory as needed
